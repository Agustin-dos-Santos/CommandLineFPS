# CommandLineFPS
A First Person Shooter at the command line? Yup...
This was originally designed for MS Windows, then ported to ncurses for Linux by BloodSharp.

## Compile instruction
- g++ CommandLineFPS.cpp -lncurses -o CommandLineFPS
